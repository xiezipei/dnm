# DNM

## What is DNM

DNM: **D**elete **N**ode **M**odules.

A simple tool to delete the node_modules directory fastly.

## Installation

```shell
npm i -g @xzp-libs/dnm
```

## Usage

```shell
cd [your_project_path]
dnm
```
